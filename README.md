2.1 中文
要在 Java 中传递名为 {name} 的字符串参数，可以在方法签名中使用 String 类型，并在调用方法时传递对应的字符串。例如：
```java
public class Example {
    public static void main(String[] args) {
        String name = "Alice";
        greet(name);
    }

    public static void greet(String name) {
        System.out.println("Hello " + name);
    }
}
```
